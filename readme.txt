== Description ==

Многоцелевой WordPress шаблон с конструктором главной страниц

== Установка ==

Вариант 1.
1. Перейдите в раздел Внешний вид - Темы - Добавить новую - Загрузить тему
2. Выберите скачанный архив и нажмите Установить
3. Активируйте тему
4. Перейдите в раздел Настройки - Лицензия и введите лицензионный ключ

Вариант 2.
1. Разархивируйте скачанный архив.
2. Поместите папку wescle в /wp-content/themes/
3. Перейдите в раздел Внешний вид - Темы и активируйте тему
4. Перейдите в раздел Настройки - Лицензия и введите лицензионный ключ
